# Vimrc of Yiguang

Mainly forked the major basic configuration of [amix](https://github.com/amix/vimrc). Used for
**Python Developing**, **Ruby Assistance** is also in plan. 

## Installation

### Local Install

```shell
$ git clone https://github.com/lukmy/vimrc.d ~/vimrc.d

$ cd ~/vimrc.d
$ ./install.sh
```
